# Juego hecho en Phaser 3 

## Pasos a seguir

* 1º Descargamos como zip y descomprimimos
* 2º Abrimos la consola
* 3º Nos metemos dentro de la carpeta descomprimida
* 4º Tecleamos el comando: <b>npm innit</b>
* 5º Tecleamos el comando: <b>npm install</b>
* Una vez finalice la instalación procedemos a iniciar el juego
* 6º Tecleamos el comando: <b>npm start</b>
* 7º En la consola si todo ha ido bien, aparecerá el puerto que debemos poner en en navegador para jugar.
